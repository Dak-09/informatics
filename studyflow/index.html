Train Your Memory in a Fun Way!

This project is a digital memory game website. Users open cards, find matching pairs, improve concentration, and enjoy a simple interactive game.

About the Project

To help users improve memory, attention, and concentration.

Interview with Adilkhan Agay

1. Why did you choose this topic?

We chose this topic because memory games are useful, simple, and interesting. They help users train their brain in a fun way.

2. Why is this project important?

This project is important because it combines education and entertainment. It shows that digital games can also be useful for learning.

3. Who can use this website?

Children, students, and adults can use this website. It is easy for different age groups.

4. What is the main aim of the game?

The main aim is to find matching pairs of cards and improve memory, attention, and concentration.

5. What technologies did you use?

We used HTML for the structure, CSS for the design, and JavaScript for the interactive game logic.

6. What part of the project was the most difficult?

The most difficult part was writing the JavaScript code for flipping cards and checking matches correctly.

7. What part of the project was the most interesting?

The most interesting part was creating the game and seeing how the cards worked on the website.

8. How does this project help students?

This project helps students improve memory skills and also shows how websites and games can be created with programming.

9. Why did you make the design colorful?

We made the design simple and colorful so it would look attractive, modern, and easy to use.

10. What result do you expect from users?

We expect users to enjoy the game, spend time usefully, and improve their memory and attention.
